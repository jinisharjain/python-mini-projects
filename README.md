Python Mini Projects

This repository contains small Python projects created to practice programming concepts such as loops, conditionals, lists, and user input. These projects help strengthen problem-solving skills and demonstrate basic application development in Python.

Projects Included
1. To-Do List

A simple command-line program that allows the user to:

Add tasks

View tasks

Exit the program

It stores tasks in a list and displays them when requested.

2. Quiz Game

A small quiz program that asks the user a few questions and calculates the final score based on correct answers.

Concepts used:

Conditional statements

User input

Score tracking

3. Password Generator

A program that generates a random password based on a length chosen by the user.

Concepts used:

Random module

Loops

String operations

Purpose of This Repository

These mini projects were created as part of my programming learning journey to move from basic syntax to small real-world applications in Python.
